# Contributing

This fork of the Open POSIX Test Suite is maintained solely for testing of [`wasi-libc`](https://github.com/WebAssembly/wasi-libc). Any tests added or modified should be done to aid development of `wasi-libc`.

This fork is a [Bytecode Alliance] project. It follows the
Bytecode Alliance's [Code of Conduct] and [Organizational Code of Conduct].
